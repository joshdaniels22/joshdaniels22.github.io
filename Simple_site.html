Inside the "main" element is content that is specific to the current webpage. The content in main should not be in any other webpage. Things that are repeated like headers and footers should not be included in "main". "main" itself should not contain text. It contains further elements that contain text.
The "header" element contains the webpage's introductory information or heading and possibly some metadata that is relevant to the content, like the date and time of the post for example. It could also contain things like a table of contents for a longer document.
This text is contained in an "article" element. It is used for things like blog posts or new articles as the name would suggest. It is used to contain content that could be distributed outside of the context of the webpage.
This text is contained in a "section" elemenet. These are used to separate different parts of the same post or article.
The "aside" element is used to provide information that is not actually part of the blog or post but is related to the article and may be considered useful to the reader>
The "footer" element contains infomation about the content like the author or copyright infomation or links to related websites.
